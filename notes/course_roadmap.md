## Week 01 - Jupyter and Python Fundamentals

- Focus: Get comfortable with notebooks, Markdown, variables, and basic Python types.
- Objectives:
- Run notebook cells and write Markdown notes
- Create variables with the main Python data types
- Use f-strings to build readable output

## Week 02 - Functions, Control Flow, and Loops

- Focus: Package logic into functions and solve small problems with conditionals and loops.
- Objectives:
- Write a simple function with parameters
- Use if conditions and for-loops together
- Generalise a one-off solution into reusable code

## Week 03 - Collections and Strings

- Focus: Manipulate lists, slices, and strings confidently.
- Objectives:
- Use list methods safely
- Slice lists and strings
- Apply common string operations

## Week 04 - Dictionaries and Nested Data

- Focus: Represent more realistic data with dictionaries, nested structures, tuples, and sets.
- Objectives:
- Create dictionary-based records
- Remove duplicates from a collection
- Write a helper that compares iterables

## Week 05 - Input, While Loops, and OOP

- Focus: Validate input, loop until a condition is satisfied, and represent things with classes.
- Objectives:
- Use a while-loop for validation
- Define a simple class
- Store objects in a list

## Week 06 - Standard Library and Environments

- Focus: Use built-in functions, explore modules, and understand conda and pip at a practical level.
- Objectives:
- Use common built-in utilities fluently
- Import and use a standard library module
- Explain the role of environments and packages

## Week 07 - Scripts, Modules, and Third-Party Libraries

- Focus: Move beyond notebooks into scripts, imports, reusable modules, and introductory data work.
- Objectives:
- Explain the difference between a notebook and a script
- Import a helper module
- Use pandas with a reusable stats helper

## Week 08 - Data Analysis Basics

- Focus: Inspect a dataset properly before modelling with head, info, describe, value counts, plots, and a train-test split.
- Objectives:
- Profile a dataset with pandas
- Spot missing values and type issues
- Create a reproducible split

## Week 09 - Correlation, Feature Engineering, and Imputation

- Focus: Look deeper into signal, create more informative features, and handle missing data cleanly.
- Objectives:
- Rank correlations with the target
- Create engineered housing features
- Impute missing numeric values

## Week 10 - Data Preprocessing Pipelines

- Focus: Encode categorical data, scale numeric features, and build consistent scikit-learn pipelines.
- Objectives:
- Compare encoding strategies
- Write a small custom transformer
- Prepare mixed data with a reusable pipeline

## Week 11 - Introduction to Machine Learning

- Focus: Understand classification, regression, and clustering through practical examples.
- Objectives:
- Identify the main ML problem types
- Train one simple model for each type
- Use a basic evaluation metric

## Week 12 - Applied Machine Learning Capstone

- Focus: Bring the course together with clean end-to-end regression and classification mini-projects.
- Objectives:
- Build a portfolio-ready workflow with preprocessing and evaluation
- Compare at least two models on the same split
- Write short conclusions about results and next steps
