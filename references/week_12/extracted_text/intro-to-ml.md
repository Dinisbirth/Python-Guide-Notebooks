Introduction to Machine Learning
Machine Learning (ML) is the study of algorithms that improve automatically through experience.
Key Concepts
:
Data
: Raw information (e.g., images, text, numbers).
Model
: A system trained to find patterns in data.
Training
: Teaching the model using data.

## Slide 4

Supervised vs. Unsupervised Learning
Supervised Learning
Definition
: Learn a function from labelled data (inputs and outputs).
Types
:
Classification
: Predict categories (e.g., "spam" or "not spam").
Regression
: Predict continuous values (e.g., house prices).

## Slide 5

Supervised vs. Unsupervised Learning
Unsupervised Learning
Definition
: Find hidden patterns in
unlabeled
data.
Types
:
Clustering
: Group similar items together (e.g., customer segmentation).
Dimensionality Reduction
: Simplify data while retaining its structure.
