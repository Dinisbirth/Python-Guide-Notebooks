# Week 04 - Dictionaries and Nested Data

**Focus:** Represent more realistic data with dictionaries, nested structures, tuples, and sets.

## Key Highlights
- Dictionaries as labelled records
- Lists of dictionaries for more realistic data
- Sets and tuples for deduplication and comparisons

## Learning Objectives
- Create dictionary-based records
- Remove duplicates from a collection
- Write a helper that compares iterables

- Prompt: Write a function that returns elements present in both iterables.
- Reinforces: Represent more realistic data with dictionaries, nested structures, tuples, and sets.
