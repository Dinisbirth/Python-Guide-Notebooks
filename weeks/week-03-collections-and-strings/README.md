# Week 03 - Collections and Strings

**Focus:** Manipulate lists, slices, and strings confidently.

## Key Highlights
- List creation, adding, removing, indexing, and slicing
- Strings as ordered sequences
- Introductory transformation challenges on text and lists

## Learning Objectives
- Use list methods safely
- Slice lists and strings
- Apply common string operations

## Source Materials
- Week 3 (1).pptx

## Practice Tasks

## Task 1: List practice

- Prompt: Create a list of names, remove one, add two, and print it in reverse order.
- Reinforces: Manipulate lists, slices, and strings confidently.

## Task 2: String challenge

- Prompt: Reverse a sentence, extract selected words, and reverse each word separately.
- Reinforces: Manipulate lists, slices, and strings confidently.
