# Week 02 - Functions, Control Flow, and Loops

**Focus:** Package logic into functions and solve small problems with conditionals and loops.

## Key Highlights
- Function definitions, arguments, return values, and type annotations
- Operators, comparisons, and if-statements
- Loop-based problem solving with multiples of 3 and 5

## Learning Objectives
- Write a simple function with parameters
- Use if conditions and for-loops together
- Generalise a one-off solution into reusable code
