# Week 01 - Jupyter and Python Fundamentals

**Focus:** Get comfortable with notebooks, Markdown, variables, and basic Python types.

## Key Highlights
- What Python is and why a high-level language is useful
- How to work with Jupyter code and Markdown cells
- Integers, floats, strings, booleans, scope, and naming

## Learning Objectives
- Run notebook cells and write Markdown notes
- Create variables with the main Python data types
- Use f-strings to build readable output
