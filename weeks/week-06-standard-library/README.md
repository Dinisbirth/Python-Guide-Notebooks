# Week 06 - Standard Library and Environments

**Focus:** Use built-in functions, explore modules, and understand conda and pip at a practical level.

## Key Highlights
- Python's batteries-included philosophy
- Useful built-ins such as zip, sorted, all, any, max, and min
- Package management with conda environments and pip installs

## Learning Objectives
- Use common built-in utilities fluently
- Import and use a standard library module
- Explain the role of environments and packages
